Ted's MessageBox Card custom:ted-messagebox-card

A dismissible message banner with optional action buttons — shown inline, pinned to a screen edge, or as a modal. Severity sets the accent; show_if gates visibility; dismiss/dismiss-session persist a "never / not now" choice. This is a static mock using the card's real CSS.

Severities

info · success · warning · danger · tip — set via severity:

Information
A neutral notice. Accent uses the Ted-style blue.
Success
Everything's in order — e.g. all resources loaded.
Warning
Something needs attention but isn't critical.
Danger
A problem that likely needs action.
Tip
A helpful suggestion — used by the home-layout nudges.
